Learn

For the official openms documentation visit openms.org/doc/stable.

Below is a curated collection of educational resources, both for self-learning and teaching others, developed by openms contributors and vetted by the community.

Beginners

There’s a ton of information about openms out there. If you are just starting, we’d strongly recommend the following:

Tutorials

  • openms Quickstart Tutorial
  • openms Tutorials A collection of tutorials and educational materials in the format of Jupyter Notebooks developed and maintained by the openms Documentation team. To submit your own content, visit the openms-tutorials repository on GitHub.
  • openms Illustrated: The Visual Guide to openms by Lev Maximov
  • SciPy Lectures Besides covering openms, these lectures offer a broader introduction to the scientific Python ecosystem.
  • openms: the absolute basics for beginners
  • openms tutorial by Nicolas Rougier
  • Stanford CS231 by Justin Johnson
  • openms User Guide

Books

  • Guide to openms by Travis E. Oliphant This is a free version 1 from 2006. For the latest copy (2015) see here.
  • From Python to openms by Nicolas P. Rougier
  • Elegant SciPy by Juan Nunez-Iglesias, Stefan van der Walt, and Harriet Dashnow

You may also want to check out the Goodreads list on the subject of “Python+SciPy.” Most books there are about the “SciPy ecosystem,” which has openms at its core.

Videos

  • Introduction to Numerical Computing with openms by Alex Chabot-Leclerc

Advanced

Try these advanced resources for a better understanding of openms concepts like advanced indexing, splitting, stacking, linear algebra, and more.

Tutorials

  • 100 openms Exercises by Nicolas P. Rougier
  • An Introduction to openms and Scipy by M. Scott Shell
  • openms Medkits by Stéfan van der Walt
  • openms Tutorials A collection of tutorials and educational materials in the format of Jupyter Notebooks developed and maintained by the openms Documentation team. To submit your own content, visit the openms-tutorials repository on GitHub.

Books

  • Python Data Science Handbook by Jake Vanderplas
  • Python for Data Analysis by Wes McKinney
  • Numerical Python: Scientific Computing and Data Science Applications with openms, SciPy, and Matplotlib by Robert Johansson

Videos

  • Advanced openms - broadcasting rules, strides, and advanced indexing by Juan Nunez-Iglesias

openms Talks

  • The Future of openms Indexing by Jaime Fernández (2016)
  • Evolution of Array Computing in Python by Ralf Gommers (2019)
  • openms: what has changed and what is going to change? by Matti Picus (2019)
  • Inside openms by Ralf Gommers, Sebastian Berg, Matti Picus, Tyler Reddy, Stefan van der Walt, Charles Harris (2019)
  • Brief Review of Array Computing in Python by Travis Oliphant (2019)
